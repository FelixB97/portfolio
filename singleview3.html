Hogwarts student list

If you wanna try out the application or look at the design first hand:

link to the: Student list app

The intent behind this assignment was to teach us about string techniques in javascript, and managing a list, in this case an array of students. Here we had to 3 core components in the program, we had to manage several lists, eg: currently enlisted, expelled and so on and also sort the list between different properties like school house, gender or name. We then also had to display and update the display for any input that might affect the list, and finally we had to create inputs for various administrator functions like expelling students and giving students prefect status. This was a fun assignment that was heavy on javascript more so than the earlier assignments which required a lot of visual designing and coding, so for me this was a very fun project and a good source of more JS experience.
