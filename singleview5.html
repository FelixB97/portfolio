Custom SNES webpage

If you wanna try out the application or look at the design first hand:

link to the: SNES Fan site

This was a really fun project to work on, we had very few mandatory needs so we where allowed a lot of creative freedom. My group and I choose to create a configurator and fan-site around the classic console the Super Nintendo Entertainment System SNES for short. We got to use NPM packages and plugins, and several different web animation solutions taught during the current theme, and i personally became a fan of GSAP scroll animations. Most fun of all we all decided on bit like design, and we created characters for each group member, one member of our group did not have time to design their own character and told us he wanted his character to be able to "dab" sarcastically. So when i was creating his character design i created some extra frames so his character would be dabbing non stop. (select Niklas for the dabbing animation)
