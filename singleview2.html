Simple JS game

If you wanna try out the application or look at the design first hand:

link to the: Simple JS game

This project was pretty much the first JS task where we had to use more than a few functions, figuring out how best to tackle and applying what was a very basic understanding of JS we each created our own simple game. My game is a type of wack-a-mole, you click the demons for points, and if you click on a human you loose 1/3 health, there is also a timer making sure the game ends if the player hasn't died from loosing all their health or won by getting 10 points. The creation of the game helped me understand many core concept of good coding, knowing how to chain functions correctly and logically, and the importance of taking your time to map out and plan how you want to code before doing it.
